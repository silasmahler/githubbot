Welcome to DO!Hack's GitHub Bot. We coordinate all submissions using a GitHub organization. To Invite you and your teammates to the organization and create a repository for you, follow these simple steps.

  1. Sign in with your GitHub account
  2. Go to "Create Repository"
  3. Fill in your repositories name (your team name)
  4. Add a description (optional)
  5. Add the Github account names of your teammates, if you are less than 5 people, leaf the other fields empty
  6. Click on "Create"
  7. Go to your GitHub profile or your email system and accept the invitations for the organization and the repository
